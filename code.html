Mikrokosmos

"Mikrokosmos" – BTS

Microcosm [noun]: something that has the same qualities as something much larger.

I got inspired by BTS' song called "Mikrokosmos." When one listens to it, they get visions of star-littered galaxies and dreams. The lyrics express BTS' desire to make their fans and casual listeners feel special.

This is a project that I did for ART101 taught by professor Steve Durie. It's a challenge for me to learn coding from scratch, though I have learned a lot from this class.
Time-based Algorithms Project requirements:
A. Millis Command
B. Include using at least 1 Algorithm
C. Custom Class
D. Create complexity that requires saveFrame()
There's no project in this class that I found not challenging. But it is only for the good! Never have I ever felt like I learn so much about art behind programming like this. Here in this project, I included the Spring algorilthm, and I used Millis Command on each of the Spring balls. I also made a custom class called Spot, made Array of it, and have it as the background. It took me a while to make it all work together and look the way I wanted. The recording process was laggy, but thanks to professor Durie for giving us some tips on adjusting "frameRate" to make the result more fluid!
